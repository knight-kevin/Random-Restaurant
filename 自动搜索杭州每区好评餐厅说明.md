# 自动搜索杭州每区好评餐厅说明

本项目提供脚本 `scripts/fetch-amap-district-restaurants.cjs`，可以使用高德开放平台 Web 服务 API，按杭州 13 个区县市搜索餐饮 POI，读取公开返回的评分字段，并按评分从高到低取每区 30 家。

## 需要准备

1. 打开高德开放平台，注册/登录。
2. 创建应用，添加 `Web服务` Key。
3. 拿到 Key 后，不要上传到 GitHub，不要写进代码。

## 运行方式

在 PowerShell 中进入项目目录：

```powershell
cd "D:\Codex\随机打卡"
$env:AMAP_KEY="你的高德Web服务Key"
node scripts\fetch-amap-district-restaurants.cjs
```

脚本会生成并更新：

- `hangzhou-district-top30-import.json`
- `restaurants.json`

然后重新上传 `_github_pages_upload` 里的文件到 GitHub Pages。

## 数据口径

- 来源：高德开放平台 POI 搜索公开返回字段
- 筛选：每个区县市搜索多个餐饮关键词，按评分从高到低取 30 家
- 字段：餐厅名、地址、分类、行政区、商圈、评分、人均/标签、来源、区内排名

注意：高德 POI 搜索不是“高德扫街榜”。它是公开 POI 搜索结果 + 评分字段整理。
