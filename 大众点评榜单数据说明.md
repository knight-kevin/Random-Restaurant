# 大众点评榜单数据说明

当前没有直接把“大众点评杭州每个区域餐厅前 30”写入餐厅库，原因如下：

1. 大众点评网页会进入美团验证中心，不能稳定公开读取榜单页面。
2. 大众点评 `robots.txt` 对 GPTBot 禁止抓取全站内容。
3. 榜单排名属于平台动态排序数据，不能用普通搜索结果冒充官方前 30。

项目已经准备好导入格式。请把你从大众点评 App 或网页手动整理出的榜单保存为 `dianping-ranking-import.json`，格式参考 `dianping-ranking-import-template.json`。

字段说明：

- `name`：餐厅名称
- `address`：尽量完整的门店地址
- `note`：推荐菜、人均、评分、榜单摘要等
- `category`：`bbq`、`hotpot`、`stir`、`noodle`、`seafood`、`beef`、`fast`
- `district`：杭州行政区，例如西湖区、拱墅区、萧山区、余杭区、钱塘区、上城区、滨江区、临平区、富阳区、临安区、桐庐县、淳安县、建德市
- `businessArea`：商圈或片区
- `source`：建议写 `大众点评区域榜`
- `rank`：该区域榜单排名，1 到 30
- `createdAt`：ISO 时间

整理好后运行：

```bash
node scripts/import-dianping-ranking.cjs
```

脚本会把数据合并进 `restaurants.json`，同名同地址的餐厅会被更新，不会重复添加。
