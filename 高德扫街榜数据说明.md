# 高德扫街榜数据说明

本项目已经预留字段，可以直接承载高德扫街榜数据：

- `name`：餐厅名
- `address`：尽量完整的高德地址
- `note`：推荐菜、人均、榜单摘要等
- `category`：`bbq`、`hotpot`、`stir`、`noodle`、`seafood`、`beef`、`fast`
- `district`：杭州行政区，例如西湖区、拱墅区、萧山区、余杭区、钱塘区、上城区、滨江区、临平区、富阳区
- `businessArea`：商圈或片区
- `source`：建议写 `高德扫街榜`
- `rank`：该区域榜单排名
- `createdAt`：ISO 时间

当前未把“高德扫街榜每区前 30”写入餐厅库，因为公开网页检索没有拿到官方完整榜单明细。高德开放平台提供 POI 搜索能力，但榜单类数据通常不等同于普通 POI 搜索结果，且需要合法接口权限或从高德 App 内手动整理。

如果后续有高德榜单截图、导出表格或 API 返回数据，可以按 `gaode-saojiebang-import-template.json` 的格式批量加入 `restaurants.json`。
